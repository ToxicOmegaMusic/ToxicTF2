Script Console Commands:
	exec fix - Runs all the commands that solve common issues in the game (invisible players, looping sound, etc).
	exec enable_shadows - Enables low quality shadows.
	exec enable_goodshadows - Enables full quality shadows (Used competetively to see players through some walls).
	exec disable_shadows - Disables shadows.
	exec enable_ragdolls - Enables ragdolls and gibs.
	exec disable_ragdolls - Disables ragdolls and gibs.
	exec antiw - "Tries" to kill the player when they press W while rocket jumping. Intended for jump maps.
	exec goodgfx - Really pretty!
	exec midgfx - Kinda meh?
	exec potatogfx - Taters!
	exec mg - Market garden script that makes right click for your melee! 

---Wanna change crosshair size or shape but it keeps resetting? You can find the options for it in the "settings.cfg"---

Also some binds:
	Q - Toggles first person viewmodels.
	R - Say "Thanks!" (Unless you're spy, then its to change the team colors in the diguise kit).
	ALT - Toggles floating health bar, use if you can't see allies health.
	CTRL - Schadenfreude if u have it equiped.
	+ (plus) - suicide key
